## 3. 핵심 기능
- **주요 기능 1** : 인증/계정
  1) 회원가입 / 로그인 / 로그아웃
  2) Password 암호화(BCrypt)
  3) 권한
  4) 마이페이지(정보 수정, Password 수정 등)
  5) 이메일 or SNS 인증 / 휴먼/ 탈퇴
- **주요 기능 2** : CRUD
  1) 목록(list) 및 CURD
  2) 페이징
  3) 검색 및 정렬
- **주요 기능 3** : 파일 / 이미지
  1) 이미지 업로드(PreView 포함)
  2) 첨부파일 업로드 / 다운로드
  3) 확장자/용량 제한, 저장 방식(Local / S3)
- **주요 기능 4** : 주문 / 결제 / 배송 

> ✍️ *필요 시 기능별로 간단한 설명을 덧붙이세요.*
