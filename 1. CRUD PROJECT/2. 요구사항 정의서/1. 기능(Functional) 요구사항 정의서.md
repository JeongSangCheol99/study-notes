# 기능 요구사항 정의서 (Functional Requirements)

## 1. 개요
본 문서는 [CRUD PROJECT]의 기능 요구사항을 정의한다.  
프로젝트에서 구현할 기능 범위를 명확히 하고, 개발 및 일정 산정의 기준으로 활용한다.

---

## 3. 범위 관리 원칙
- `필수 = Y` 인 기능만 1차 MVP 범위로 구현한다.
- `선택` 기능은 2차 개선 범위로 분리한다.
- 신규 기능 추가 시 반드시 본 문서에 먼저 정의 후 개발한다.
